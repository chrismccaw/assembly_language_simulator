Supported Structures

IF STATEMENT

int x = 5;
if(x < 6){
x = 10;
}

WHILE LOOP:

int x = 0;
while(x <= 5){
x++;
}

EXPRESSIONS:

int x = 1 + 2;

FUNCTION CALLS:

A main method, main(). Is required to call the function.

int foo(int x, int y){
int y = x + y;
}

int main(){
foo(1,2);
}

Support Primitive Types: int
